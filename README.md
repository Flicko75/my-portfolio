Personal portfolio website for a backend focused developer.

## Live Website :
https://sayan-sarkar.netlify.app/

## Tech Stack

- Vite
- TailwindCSS

## Projects Included

### To run locally -
```
git clone https://github.com/Flicko75/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```
